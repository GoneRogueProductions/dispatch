Dispatch 2.0 is out!

1st January 2023

After a month of tireless work, Dispatch 2.0 is out! From this new and improved news page to everything else, Dispatch has had a leg-up!

Here are all the new features.

  • New landing page

  • New login page w/ slideshow

  • Bring back the new chat interface

  • MORE JAVASCRIPT

  • Chats now use AJAX for faster sending w/ no redirects

  • Updated upload prompt

  • User Lookups

  • Cookies Policy

  • Reporting Function

  • More Administrator features

  • The great groups overhaul

  • Codeteam page

  • User Descriptions

The AJAX was really great, and the code allows for our (laggy) free servers tp have a bit less stress, with a faster sending time for the user.

We hope you have a great New Year, as we have.

@max and @werdl
